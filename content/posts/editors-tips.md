## VS Code

### Tips

#### Disable Hardware Acceleration

- Open the Command Palette [Ctrl + Shift + P]
- Run the `Preferences: Configure Runtime Arguments` command
- Add `"disable-hardware-acceleration": true`
- Restart VS Code
